「うみがめのスープ」

最初に読んでね．

・コマンドの使い方
  "/umigame" または "/u" でコマンドが使えるよ
  "/umigame help" で見てね

・問題ファイルの用意
  plugins/Umigame/Questions の中に問題文を，
  plugins/Umigame/Answers の中に回答文を 書いた.txtファイルを入れてね．ファイル名は日本語でも動くよ．
  入れたらゲームを開始すると自動でコマンドの予測変換に出てくるようになるよ．便利だね．
  ファイル名が "_" から始まるファイルは予測変換に追加されないように設定されてるよ．有効活用してね．

・ゲームの進め方
  1. 問題ファイルを用意してね．
  2. 参加者ごとに "/umigame pinpon <プレイヤー名>" を入力したコマンドブロックを用意してね．ボタンをつけてあげよう．
  3. 参加者が揃ったら "/umigame start" でゲームを開始しよう．
     GMを指名したい時は "/umigame start <GMのプレイヤー名>" で指名できるよ．
     指名なければコマンド実行者がGMになるよ．
  4. 参加者にはボタンを押して回答してもらおう．1秒以内の同時押しは順番表示が出るよ．
  5. いい質問だなって思ったらエメラルドブロックで，正解！ってなったら金ブロックでその人をぶん殴ろう．
     ちなみにこのプラグインが入ったサーバーではダメージ，満腹度は常時無効化されるよ．
  6. ゲームが終わったら "/umigame end" でゲームを終わらせよう．最終結果が表示されるよ．ランキングは回答ポイントのみで判定されます．
  7. 次のゲームへ行くときは一度サーバーをリロードしてね．その時に問題解答ファイルを入れ替えるのを忘れずにね．

・注意点
  ゲーム中にリログされるとバグるかも．一応対策したけど動くかよくわかんない．
  GMに得点与えようとするとこれまたバグるよ．やめよう．
